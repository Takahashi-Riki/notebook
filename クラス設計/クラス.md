## インスタンス変数

- コンストラクタで確実に正常値を設定しているか？ ※1
- 不変になっているか？ ※1

# サンプルコード

## サンプル1

```ruby
class Money
  # インスタンス変数を不変にする。
  attr_reader :amount, :currency

  def initialize(amount, currency)
    # コンストラクタで確実に正常値を設定する。
    raise ArgumentError.new("金額が0以上でありません") if amount < 0
    raise ArgumentError.new("通貨を指定してください") if currency.nil? || currency.empty?
    @amount = amount
    @currency = currency
  end
end
```
